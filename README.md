# Gym Memory Bot

Телеграм-бот для трекинга тренировок. Читает сообщения тренера и атлета в групповом чате, парсит упражнения, сохраняет в базу, строит графики прогресса.

## Структура файлов

```
bot.py        — основной файл, хендлеры команд
parser.py     — парсинг сообщений тренера и атлета
database.py   — SQLite база данных
charts.py     — генерация графиков
requirements.txt
railway.toml
```

## Деплой на Railway

### 1. Подготовка репозитория

```bash
git init
git add .
git commit -m "init"
```

Создай репозиторий на GitHub, залей туда:
```bash
git remote add origin https://github.com/ВАШ_USERNAME/gym-bot.git
git push -u origin main
```

### 2. Railway

1. Зайди на [railway.app](https://railway.app) → Sign in with GitHub
2. New Project → Deploy from GitHub repo → выбери репозиторий
3. После создания проекта: Variables → Add Variable:
   - `BOT_TOKEN` = твой токен от BotFather
   - `TRAINER_USERNAME` = username тренера в Telegram (без @)
   - `DB_PATH` = `/data/gym.db`
4. Settings → Add Volume → Mount Path: `/data` (постоянное хранилище для базы)
5. Deploy → бот запустится автоматически

### 3. Проверка

В Telegram: `/start` в личке боту.

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/exercises` | Все упражнения в базе |
| `/progress жим` | График по упражнению (всё время) |
| `/progress жим месяц` | График за месяц |
| `/stats` | Сводка за 30 дней |

### Периоды для /progress
`неделя` · `месяц` · `квартал` · `полгода` · `год`

## Как пользоваться

1. Создай группу в Telegram
2. Добавь бота и тренера в группу
3. Тренер пишет план → бот парсит автоматически
4. Ты пишешь результат (с X/10 после упражнений) → бот сохраняет
5. `/progress становая тяга квартал` → получаешь график

## Импорт истории

Чтобы загрузить старые тренировки: перешли сообщения из старого чата в новый групповой чат с ботом. Бот распарсит их по оригинальной дате пересылки.

> Для точного импорта с оригинальными датами — напиши, сделаем отдельный скрипт под экспорт Telegram.
